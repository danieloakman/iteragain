      • <T, Length>(arg: IteratorOrIterable<T>, length: Length, offset: number, fill?: T): ExtendedIterator<Tuple<T, Length>>

      • Yields sliding windows (tuples) of length from the input arg. Each window is separated by offset number of elements.

        Example

        windows([1,2,3,4,5], 2, 1).toArray() // [[1,2], [2,3], [3,4], [4,5]]
        windows([1,2,3,4,5], 2, 3).toArray() // [[1,2], [4,5]]
        windows([1,2,3,4,5], 3, 3, 0).toArray() // [[1,2,3], [4,5,0]]

        • offset: number
